# VaultFi – Fixed-Rate DeFi Savings Vault

VaultFi is a decentralized finance (DeFi) protocol that demonstrates how fixed-rate savings behavior can be implemented using smart contracts.

The protocol allows users to deposit ERC-20 tokens into a smart contract vault and earn predictable time-based yield.

## Features

- ERC-20 token deposits
- Fixed interest rate
- Time-based interest calculation
- Withdrawal of principal plus interest
- Transparent on-chain financial rules

## Project Structure

contracts/VaultFi.sol – main smart contract  
scripts/deploy.js – deployment script  
test/VaultFi.test.js – testing files
